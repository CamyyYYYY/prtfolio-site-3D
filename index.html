Portfolio Project 1: Fully Autonomous Kissing Robot Fully autonomous kissing robot Used python for the kissing app (which I made via vs code on my laptop) The turned it into a linux executable so it could run on the raspberry pi board (the board is attached behind the screen. I bought the screen online, and made sure it was compatible w/ my specific raspberry pi, also the package came w/ instructions for how to connect it) Setup a separate Arduino project using an Arduino Uno, a servo motor, a sensor, and another board attached on-top of the Arduino (more specifically an L293D Drive Module Motor) Soldered header pins to the L293D In order to get some wires connected from the sensor and the servo motor Got some motors and wheels from an Arduino kit I found online In order to setup the robots movement, and attached those wires to the L293D as well After all was said and code was pushed to the Arduino Uno, I connected the raspberry pi ONTOP of the actual moving Arduino robot via hot glue, I also added some supporting axles at the bottom to make sure the robot could support Its own movement Made It so the sensor moves from left to center to right to sense any objects before backing up out of the objects way, and slightly turning right Sound Is quiet since I used some quiet headphones but the linux executable asks the user for a kiss, and when the user pressed the button presented, a “thank you” message appears and the app says “thank you so much” Lastly I used a power bank (INIU Portable Charger w/ the paw print) to power both devices via their respect power cables Followed this tutorial for the Arduino robot (my source code behaves slightly different, as I wanted to optimize things more towards my preference) Followed this tutorial for the python app Self made the linux executable
